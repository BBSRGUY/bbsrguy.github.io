About ODEN

ODEN (Odia-English Neural model) is a state-of-the-art bilingual language model designed to process and generate text in both English and Odia languages seamlessly.

By bridging linguistic barriers, ODEN opens new possibilities for communication, education, and cultural preservation.

Model Architecture

Input Embedding

Transforms tokens into dense vector representations that capture semantic meaning.

Encoder Layers

Contextualizes input text through multiple transformer layers using self-attention.

Cross-Attention

Facilitates translation and reasoning by attending to different parts of the input sequence.

Decoder Layers

Generates coherent output text based on encoded representations.

Output Projection

Converts model output into probability distributions over vocabulary tokens.
